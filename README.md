# My Diary

書いている最中の手触りを大事にした、自分用の日記サイトです。

## 使い方

`index.html` をブラウザで開くと使えます。

ローカルサーバーで見る場合:

```powershell
python -m http.server 5174 --bind 127.0.0.1
```

そのあと `http://127.0.0.1:5174/index.html` を開きます。

## 保存

日記はブラウザの `localStorage` に自動保存されます。外部サービスには送信しません。

## できること

- 入力中の自動保存
- 文字数、行数、連続日数の表示
- 気分色の切り替え
- 前日/翌日への移動
- 月ごとのカレンダー表示
- 本文やタグの検索
- よく使うタグのクイック絞り込み
- 表示だけをぼかす、のぞき見防止モードと自動ぼかし
- 今日の日記を空にした直後の取り消し
- 一度開いたあとのオフライン表示
- 1日分の Markdown 書き出し
- 全日記の JSON バックアップと復元

## キーボード操作

- `Ctrl/Cmd + S`: 保存
- `Alt + ← / →`: 前日/翌日へ移動
- `Esc`: のぞき見防止
- `Ctrl/Cmd + Enter`: Markdown 書き出し
