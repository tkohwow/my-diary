# My Diary

書いている最中の手触りを大事にした、自分用の日記サイトです。

## 使い方

`index.html` をブラウザで開くと使えます。

ローカルサーバーで見る場合:

```powershell
python -m http.server 5174 --bind 127.0.0.1
```

そのあと `http://127.0.0.1:5174/index.html` を開きます。

## 保存

日記はブラウザの `localStorage` に自動保存されます。外部サービスには送信しません。

## できること

- 入力中の自動保存
- 文字数、行数、連続日数の表示
- 気分色の切り替え
- 月ごとのカレンダー表示
- 本文やタグの検索
- 表示だけをぼかす、のぞき見防止モード
- 一度開いたあとのオフライン表示
- 1日分の Markdown 書き出し
- 全日記の JSON バックアップと復元
